# Student Management System (Python)

A beginner-friendly Python project to manage student details with JSON storage.

## Features
✅ Add Student  
✅ View Students  
✅ Search Student  
✅ Delete Student  
✅ Data stored in `students.json`

## How to Run
1. Download this repository
2. Run the program:
   python main.py

## Sample Menu
1. Add Student  
2. View Students  
3. Search Student  
4. Delete Student
